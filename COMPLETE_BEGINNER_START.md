## ü§î What is Programming?

### In Simple Terms:

**Programming** = Writing instructions for a computer to follow

**That's it!** Computers are very fast but very dumb. They can only do **exactly** what you tell them.

**Real-world analogy:**

Think of a recipe:
```
1. Crack 2 eggs into bowl
2. Add 1 cup of flour
3. Mix for 2 minutes
4. Pour into pan
5. Bake at 350¬∞F for 20 minutes
```

This is a **program** for making a cake! Each step is an instruction.

**Python code is similar:**
```python
1. Get user's name
2. Create greeting message
3. Display message on screen
```

## üß† Key Concepts for Absolute Beginners

### 1. Variables (Storage Boxes)

**Concept:** A variable stores a value with a name

**Real world:**
- You have a box labeled "age"
- You put the number 25 inside
- When you need your age, you check the box

**In Python:**
```python
age = 25
```

This creates a "box" called `age` and puts `25` in it.

---

### 2. Data Types (Different Kinds of Data)

Just like real world has different types of things:
- Numbers (can do math with)
- Text (can't do math with)
- True/False (yes or no questions)

**Python has data types:**

```python
# Number (integer)
age = 25

# Number (decimal)
price = 19.99

# Text (string)
name = "Josie"

# True/False (boolean)
is_student = True
```

---

### 3. Functions (Reusable Recipes)

**Concept:** A function is a named set of instructions you can use over and over

**Real world:**
- Recipe for cookies
- Recipe for pizza
- Recipe for pasta

You don't rewrite the recipe every time - you just say "make cookies!" and follow that recipe.

**In Python:**
```python
def make_greeting():
    print("Hello!")
    print("Welcome!")

# Use it (call it)
make_greeting()  # Runs both print statements
```

---

### 4. Input/Output (Talking to the Computer)

**Input:** Data going **into** your program
- User types something
- File is read
- Data from internet

**Output:** Data coming **out** of your program
- Display text on screen
- Save to file
- Send over internet

**Example:**
```python
# Output (computer shows you this)
print("What's your name?")

# Input (you type an answer)
name = input()  # You type: Josie

# Output (computer responds)
print("Hello, Josie!")
```

## üìñ Glossary for Beginners

**Common terms you'll see:**

| Term | Simple Definition | Example |
|------|-------------------|---------|
| **Code** | Instructions written in Python | `print("Hi")` |
| **Execute/Run** | Make the computer follow the instructions | Click "Run Tests" |
| **Function** | A named recipe of instructions | `def greet():` |

| **Integer** | Whole number | `42` |

| **Boolean** | True or False value | `True` |
| **Parameter** | Input that a function receives | In `greet(name)`, `name` is parameter |

| **Error** | When something goes wrong | `SyntaxError`, `NameError` |
| **Bug** | A mistake in your code | Using `Print` instead of `print` |
| **Debug** | Finding and fixing bugs | Reading errors, testing |
