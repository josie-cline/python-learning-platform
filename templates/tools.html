üìã Quick Reference

Common commands you'll use every day

Terminal

cd folder - Go into folder
ls - List files
pwd - Current location
mkdir name - Create folder
python3 file.py - Run Python

Git

git status - Check changes
git add . - Stage all
git commit -m "msg" - Save
git push - Upload
git pull - Download

Cursor/VS Code

Cmd/Ctrl + S - Save
Cmd/Ctrl + F - Find
Cmd/Ctrl + P - Quick open
Cmd/Ctrl + / - Comment
Ctrl + ` - Terminal

Python

python3 --version - Check version
pip3 install pkg - Install package
python3 -m venv env - Virtual env
python3 - Interactive mode
